# Example Assets for Template Guide

This folder contains example videos and images used in the Template Creation Guide.

## Required Files

### Videos
- `veo-shorts-v1-example.mp4` - Example output from the Veo Shorts template (17 seconds, 720x1280, H.264)

### Images (Optional)
- Template preview thumbnails
- Screen capture examples

## How to Add Example Video

1. Place your example video file in this folder
2. Name it: `veo-shorts-v1-example.mp4`
3. Ensure it meets specs:
   - Resolution: 720x1280px (9:16 vertical)
   - Format: MP4 (H.264 codec)
   - Duration: ~17 seconds
   - File size: Under 50MB for web performance

## Placeholder Video

If no video is present, the Template Guide will show a placeholder with template information.

You can use a sample video from:
- Google Veo examples
- Sample vertical video from stock footage
- Your own rendered template output

## File Location

These files are served from `/examples/` in the web app.

Example: `/examples/veo-shorts-v1-example.mp4`
